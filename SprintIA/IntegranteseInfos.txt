Este projeto contém dois modelos de Machine Learning treinados a partir de um dataset simulado (mock) sobre teleconsultas.

1.  modelo_regressao.joblib:
    -   Objetivo: Prever a frequência (número de dias) que um paciente faltará ao teleatendimento.
    -   Tipo de Modelo: Regressão (LinearRegression).
    -   Features Usadas: 'idade', 'distancia_km', 'historico_faltas_total'.

2.  modelo_classificacao.joblib:

    -   Tipo de Modelo: Classificação (DecisionTreeClassifier).

API REST (Arquivo app.py):
-   O modelo 'modelo_classificacao.joblib' foi disponibilizado em uma API REST usando Flask.
-   A API recebe um JSON com os dados do paciente (dias_faltas, distancia_km, historico_faltas_total).
-   Ela retorna a previsão da faixa etária (ex: {"previsao_faixa_etaria": "Jovem Adulto"}).
